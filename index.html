Robot Chef

This was definetly an experience.
  • My team and I built a robot that cooks 5 different items of food using arduino and Embedded C.
  • We used relays, different kinds of motors, and had an input system with a screen and buttons for the user to choose the item they wanted. Each item had a variable like amount of sugar for drinks or whether the user wanted cream or orzo soup.
  • We implemented it with minimum cost in mind and focused more on getting the best out of everything we used by thinking out of the box.

Logic Circuit
Simulator

Who knew connecting wires was going to be this hard.
  • My team and I designed a logic circuit Simulator using C++ and a graphics library.
  • We applied various OOP concepts like Polymorphism, Inheritance and Class Relations while taking into consideration encapsulation.
  • The design mode had features like inserting, copying,cutting, pasting and deleting (gates,lamps,switches and wires). The simulation mode had the options of Probes,Circuit Validation ,Circuit simulation with results that could be seen using lamps or truth tables. .
